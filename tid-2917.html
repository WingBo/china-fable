2005年我整理过部分,见"寓言文摘""名家名作"专栏:

中国当代寓言作家简介

 何公超（1904-1986），原名何味辛，上海市松江县人。中国当代著名儿童文学作家。原中国作家协会会员、中国寓言文学研究会会员。主要作品有《何公超童话寓言选》、《快乐岛》、《丑小鸭》、《小金鱼》等。

 陈伯吹 (1906-1997)，原名陈汝埙 ，上海市宝山县人。中国当代著名儿童文学作家。曾任中国作家协会顾问、中国寓言文学研究会顾问等。著译丰硕，主要作品有《陈伯吹文集》（四卷本）等。曽获第二届全国少年儿童文艺创作荣誉奖。

 申均之（1912-1988），山东省烟台市人。原中国作家协会会员。主要作品有小说《冬天的故事》等。著有寓言集《西郭先生》。

 严文井（1915-），原名严文锦，湖北省武昌县人。中国当代著名作家。曾任中国作家协会书记处书记、中国寓言文学研究会顾问。著作丰硕，主要作品有《南南和胡子伯伯》、《严文井童话寓言集》等多种。主编《幼学寓言百篇》等。

 金 近 (1915-1989),原名金知温，浙江省上虞县人。中国当代著名儿童文学作家。曾任中国作家协会理事等。主要作品有《春风吹来的童话》、《金近作品选》等。曽获第二届全国少年儿童文艺创作荣誉奖。

 秦 牧 (1919-1992),原名林觉夫，广东省澄海县人。中国当代著名作家。曾任中国文联委员、中国作家协会理事等。著作丰硕，主要作品有散文集《花城》、文论集《艺海拾贝》、儿童文学集《在化妆晚会上》等。

 吴梦起（1921-），笔名吴扬，山东省烟台市人。中国作家协会会员。著有中长篇小说多部，著有《吴梦起百篇童话集》等。《老鼠看下棋》获全国首届儿童文学奖；《小雁归队》获全国第二届少年儿童文艺创作三等奖；《吴梦起童话选》获宋庆龄第四届儿童文学二等奖。

 湛 卢（1922-1987），本名王文琛，四川省夹江县人。中国当代著名寓言作家。曽任中国寓言文学研究会理事。著有寓言集《猴子磨刀》、《狐狸审案》、《审判伊索的寓言》、《乌鸦开画展》等。《一帆风顺》（原名《车与船》）入选新加坡小学课本。


 任明耀（1922-），笔名任斐然，浙江省杭州市人。中国作家协会会员、中国寓言文学研究会会员、中国戏剧家协会会员。著有童话集《狐狸和灰狼》、《猪八戒新传》等。

 金 江（1923-），原名金振汉，浙江省温州市人。中国当代著名寓言作家。中国作家协会会员、中国寓言文学研究会名誉副会长。寓言著作、编著丰硕，主要作品有《金江文集》（四卷本）、《金江寓言选》等数十种。编有《中国现代寓言选》等多部。寓言《乌鸦兄弟》曾被选入师范学校语文课本；童话《白头翁的故事》曽被选入小学语文课本。1980年寓言《乌鸦兄弟》获全国第二届少年儿童文艺创作三等奖；1982年《寓言百篇》获全国优秀儿童读物奖。

 陈 模（1923-），笔名尔东，江苏省泰兴县人。中国当代著名儿童文学作家。中国作家协会会员、中国寓言文学研究会名誉副会长。著有中长篇小说多部，主要作品有《陈模儿童文学选》、《陈模童话故事选》等。著有寓言集《寓言故事一百篇》、《儿童寓言故事》等。1980年长篇小说《奇花》获全国第二届少年儿童文艺创作二等奖等。寓言剧《生死之谜》获第十一届陈伯吹儿童文学奖。

 邬朝祝（1923-），湖南省新化县人。中国作家协会会员、中国寓言文学研究会会员。著有儿童文学集多种，主要作品有《外星猫传奇》、《神秘的小奇人》等。童话《快乐的菜花蛇》获第十届陈伯吹儿童文学奖。

 黄永玉（1924-），土家族，湖南省湘西县人。中国当代著名画家、诗人。著有《罐斋杂记》《永玉三记》等。

 鲁 兵（1924-），浙江省金华市人。中国作家协会会员、中国寓言文学研究会会员。编审。儿童文学著作甚多。著有寓言集《寓言的寓言》等。

 韶 华（1925-），原名周玉铭，河南省滑县人。中国当代著名作家。曾任中国作家协会书记处书记、中国寓言文学研究会名誉会长。著作丰硕，著有中长篇小说多部，寓言集有《风筝和雄鹰》、《新聊斋夜话》等

 龙世辉（1925-1991），侗族，湖南省武冈县人。原中国作家协会会员。著有《龙世辉寓言集》等。《女娲与苍蝇》获中国寓言文学研究会首届“金骆驼奖”创作优秀成果奖。

 彭文席（1925-），浙江省瑞安县人。担任小学教师时创作的寓言《小马过河》被选入小学课本，并获全国第二届少年儿童文艺创作一等奖。

 刘 征（1926-），本名刘国正，北京市人。中国当代著名寓言诗人。中国作家协会会员、中国寓言文学研究会名誉副会长。编审。著作丰硕，著有寓言诗集《海燕戒》、《春风燕语》、《刘征寓言诗》等。《春风燕语》获全国第二届优秀诗集奖。

 徐强华（1927-），原名徐象烈，浙江省永嘉县人。中国作家协会会员、中国寓言文学研究会会员。副教授。著有寓言集《黄莺和鹦鹉》、《菩萨出汗》、《谈狗色变》等多种；编有《中国科学寓言选》等。《床下钓鱼》获中国寓言文学研究会首届“金骆驼奖”创作优秀成果奖。

 吕德华（1927-），笔名端木青枝，河北省晋县人。中国作家协会会员、中国寓言文学研究会常务理事。编审。儿童文学著作甚丰，著有寓言集《蜗牛搬家》、《月亮为什么害羞》、《狐狸吹牛》等。寓言《蜘蛛、蚕和老桑树》获第二届全国少年儿童文艺创作三等奖；《独木桥上的狐狸和狼》获中国寓言文学研究会首届“金骆驼奖”创作二等奖。

 鲁 芝（1927-2004），笔名大曼，山东省烟台市人。中国作家协会会员、中国寓言文学研究会会员。著有寓言集《狐狸和狐狸》等。《狐狸和狐狸》获中国寓言文学研究会首届“金骆驼奖”创作优秀成果奖。

 杜 梨（1927-2004），原名杜永祥，河南省南阳县人。中国寓言文学研究会会员。著有寓言集《鲜花与蒺藜》、《馋鱼赞饵》等。《馋鱼赞饵》获中国寓言文学研究会第二届“金骆驼奖”创作优秀成果奖。

 朱靖华（1928-），山东省安邱县人。中国作家协会会员、中国寓言文学研究会名誉副会长。教授。学术著作丰硕，主要作品有《苏轼新论》、《苏东坡寓言评注》等，编有《先秦寓言选译》、《历代寓言选》（上、下）、《古代寓言精华》等多种。

 马 达 (1928-), 笔名力牧,湖南省华容县人 。中国作家协会会员、中国寓言文学研究会名誉副会长。编著甚丰，编有《古代劝学寓言》、《中国古代动物寓言诗选》等多种。《中国古代寓言精华》获中国寓言文学研究会第二届“金骆驼奖”二等奖。《中国古代动物寓言故事》获第二届全国图书“金钥匙”纪念奖。

     纪 路（1930-），本名周开桃，山东省即墨市人。中国寓言文学研究会会员。著有寓言集《猴子养花》、《老虎戴铃铛》、《老鼠算卜》等。

 林颂英（1930-），上海市人。中国寓言文学研究会会员。克服身残的困难长期坚持儿童文学写作。主要作品有寓言集《老鼠和罐头》、《林颂英科学童话选》等多种。

 仇春霖（1930-），笔名舸夫，江苏省盐城市人。中国当代著名科普作家。中国作家协会会员、中国寓言文学研究会名誉会长、中国科普作家协会副理事长。教授。著作甚丰，著有寓言集《猴子刺虎》、《无花果》、《帆和舵》等。主编有《当代中国寓言大系》、《古代中国寓言大系》、《外国寓言大系》等。《叶绿花红》获国家第四届图书一等奖。

 白忠懋（1930-），浙江省镇海县人。中国寓言文学研究会会员、中国科普作家协会会员。著有科学寓言集《寓言马戏班》、《动物寓言》等。

 王树功（1931-），四川省夹江县人。中国寓言文学研究会会员。

 余毅忠（1931-），广东省中山县人。中国寓言文学研究会会员。著有寓言集《鲤鱼告状》、《老猪学吹唢呐》等。

 胡鹏南（1931-），上海市人。著有寓言集《八哥学乌鸦》、童话集《小公鸡除虫》、《魔椅子》等。主编《中外寓言大王》。

 王禄鑫（1931-），浙江省黄岩县人。中国寓言文学研究会会员。著有寓言集《楝树和乌鸦》等。

 黄瑞云（1932-），笔名胡天雁，湖南省娄底市人。中国当代著名寓言作家。中国作家协会会员、中国寓言文学研究会名誉副会长。教授。著作丰硕，主要作品有《老子本原》、《诗苑英华》（五卷本）等多种。寓言集有《黄瑞云寓言》、《答里﹒坎曼尔的幽默》等。《黄瑞云寓言》获中国新时期优秀少儿读物一等奖、中国寓言文学研究会首届“金骆驼奖”一等奖。

 陈乃祥（1932-），浙江省诸暨县人。中国作家协会会员、中国寓言文学研究会会员。副编审。著有寓言集《喜鹊嘲牡丹》、《猴子献眼镜》、《牧师与猎人》等十多种。《寓言百篇》获中国寓言文学研究会首届“金骆驼奖”创作优秀成果奖；《老虎跳崖》获第二届金江寓言文学奖。

 杨明火（1932-），浙江省宁海县人。中国作家协会会员、中国寓言文学研究会理事。主要作品有长篇儿童小说《特派员的儿子》等，著有寓言集《满目梦》、童话集《小马找撇》等。

 解普定（1932-），浙江省黄岩县人。中国寓言文学研究会理事。著有寓言集《乌龟爬天梯》、《猫头鹰和小花狗》、《银色的小河》等。《乌龟爬天梯》获中国寓言文学研究会首届“金骆驼奖”创作优秀成果奖。《秃鸦》获第三届金江寓言文学奖。

 倪树根（1933-），浙江省富阳县人。中国作家协会会员、中国寓言文学研究会会员。副编审。著有童话集《甜葡萄王国里发生的怪事》等二十多种。《月亮上飞来的歌声》获1990年陈伯吹儿童文学奖。

 刘厚明（1933-1989），北京市人。中国当代著名儿童文学作家。曾任中国作家协会儿童文学委员会副主任、中国戏剧家协会会员。著作丰硕，主要作品有剧本《箭杆河边》、《儿童喜剧选》、《绿色钱包》等多种，童话集《阿城的龟》、《耍蛇少年》等。剧本《小雁齐飞》获第二届全国少年儿童文艺创作一等奖；《绿色钱包》获1981年文化部优秀儿童故事片奖。

 蔡振兴（1934-），上海市人。中国寓言文学研究会会员。著有寓言集《乌龟查辞典》等。小说集《野人谷斗狼》获1988年陈伯吹儿童文学奖优秀作品奖。

 柯玉生 (1934-), 笔名木可,湖北省大冶市人。中国作家协会会员、中国寓言文学研究会常务理事、中国儿童文学研究会副秘书长。编著丰硕，主编《童话》丛刊、《中国新时期寓言选》、《中国当代优秀童话选》、《外国著名童话》等多种。著有寓言集《猴子抢功》。

    佟希仁（1935-），满族，笔名金盏，辽宁省抚顺市人。中国作家协会会员、中国寓言文学研究会会员。教授。儿童文学著作丰硕，主要作品有儿童诗集《雪花姑娘》、《孔雀和白头翁》、童话集《浑河岸边的明珠》等多种。散文《花的闹市》获第十届陈伯吹儿童文学奖。

 朱 锵（1935-），原名朱茂昌，浙江省温州市人。中国寓言文学研究会会员。著有寓言集《老虎打武松》、《朱茂昌童话寓言选》等。

 李仁晓（1935-），山东省安邱县人。中国作家协会会员。儿童文学著作甚多，主要作品有《李仁晓作品选》等多种。

 艾海提.阿西木（1935-），维吾尔族，新疆维吾尔自治区喀什市人。中国民间文艺家协会会员。编审。著有寓言集《鹌鹑与喜鹊》、《老鼠的忧伤》等。编有《维吾尔寓言》等。

 胡树化（1936-），原名刘德垠，江苏省铜山县人。中国寓言文学研究会会员。著有寓言集《弄蛇者和眼镜蛇》等。《弄蛇者和眼镜蛇》获中国寓言文学研究会首届“金骆驼奖”创作优秀成果奖。

