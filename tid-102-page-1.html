　　要拜师学艺的猪

　　一头猪经常受一只野狼欺负。它决定拜师学艺，好对付野狼。

　　拜谁呢？猪一动脑子，就决定拜狗为师。

　　它找到狗，说明了来意。

　　狗问猪：“你为什么要拜我为师？”

　　“因为你比狼厉害呀！”

　　“哦，----可是，还有比我更厉害的，你找它不是更好吗？”

　　“还有谁比你厉害？”

　　“老虎呀----你找它，准没错！”

　　猪一听，就找老虎去了。可是老虎又说狮子比自己厉害，要猪去找狮子；猪找到狮子，狮子又说自己也有客星，那就是大象。

　　没办法，猪只好找到大象，说明了来意。

　　大象一听，忙说：“哎呀，你别看我是庞然大物，能斗得过威武的狮子，可我对你们猪那咕隆咕隆的叫声害怕得要命……

　　“这么说，我还得拜自己为师不成？”猪一听，立即打消了拜师的念头，哼哼叽叽的离开了。


我是一只猪，没有人敢说我不是一只猪。
 那些说我是猪的人自己就很猪，而那些不认为我是猪的人也未必不猪。
 我认为我是一只猪，因为我是猪，所以我不猪。
 每天，我在城市行走，没有人知道我是丘比特手下的一只猪。
 累的时候，我会喜欢做一只死猪。或者化身为一只储钱罐，我答应过主人，他每往我口中放一枚硬币，我就给他一个美好的祝福。等我为他许下了一千零一个愿望时，我就不再是死猪。
 可是我明白，主人的女儿会将我砸了，拿里面的银子去买棒棒糖。我想我不能做一只死猪，不然我自己就太猪了。


 于是，我一直做着不是死猪的猪。


 看见秋水叮咚的时候，我依然是一只猪。


 你说我是猪吗？
 我今天心情很好。
 你认为我是猪吗？
 我刚刚吃完饭。


 这是我和他第一次对话，我知道他不是猪，而且一点都不猪。我喜欢和他在一起，他喜欢穿黑色的T恤，而我是一只白皮猪，我们总会在一起散步，他显得越黑，我显得越白。
 我知道那些说秋水叮咚是女生的人一定很猪，而那些说秋水叮咚不是女生的人也不一定不是猪。
 我是一只猪，无论我怎么认为，我都不猪。


 秋水叮咚喜欢的人是小可。我认为他的选择很正确，一点都不猪。
 小可是很好的女生，一点都不猪。在她的面前我却显得有点猪。


 每天，我和秋水叮咚走路的时候，他都说他正想着小可。他说他也许很猪，但是我觉得他一点都不猪。


 今天天气不错。
 我做完作业了。
 今天晚上有流星雨。
 我想见小可。


 每天，我和秋水叮咚的对话重复着，我们都能好好地交流，因为我们不猪。


 秋水叮咚一直不知道我是丘比特养的猪。他只是希望小可找到她所向往的大海时，他在海的对岸等着她。
 我想对小可说，秋水叮咚会一直等你，一直爱你。
 我想对小可说，当你找到你喜欢的那片海的时候，你会看到一个人在等你，那个人不猪。
 你还会看到一只猪，一只爱情白皮猪。
